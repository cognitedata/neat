**Neat supports more than 10 validation rules** for data modeling. These rules are learned from
best practice, knowledge of the Cognite Data Fusion data modeling service, and practical experience from
helping customers build and maintain their data models.



# Connections (NEAT-DMS-CONNECTIONS)

| code | name                                                   | message                              |
|------|:-------------------------------------------------------|--------------------------------------|
| 1001 | [Required Field](https://example.com/required-field)   | This field is required.              |
| 1002 | [Invalid Format](https://example.com/invalid-format)   | The format is invalid.               |
| 1003 | [Value Too Large](https://example.com/value-too-large) | The value exceeds the allowed limit. |
